== Description ==

Have you ever felt stranded from sharing any blog/post/feed which you have liked and want to share it with your followers? You must have. Well, its time to change it!

Going by the trend, if you want to share any type of content on Twitter without login to it each and every time, then here it is! **MWB Twitter toolkit for wordpress -Tweets, Feeds, Timeline, posts** will surely help you out this. 

Through this extension, you can surely share and follow any blog/post/feed which you like and want to get associated with it. So, if you don't want to miss the chance to read and just move on, this extension will be helpful to both reader/blogger and the administrator to make a mark. 

== Exclusive Component == 

Amplify your follower's base with this extension which will enable you to share all your tweets directly on your website related to the new blog and trending stories published by the administrator.  

== Benefits ==

* This will build an affiliation between following and sharing of the post/feed which will later build a strong client base.
* This will help to manage your profile image and banner image with ease.
* This will save the time of the blogger/reader who can directly share the post/feed from the WordPress webpage.
* This is a complete toolkit for the administrator to manage this post/feed and make it socialize on twitter. 


== Features ==

* The administrator can show the timeline of the tweet/feed with ease.
* The administrator can also show and list the top 5 tweet done by him 
* The administrator can allow the follow button for the followers. 
* The administrator can directly tweet from the backend
* This will allow the reader/blogger to share blog content/blog directly from the page with ease.
* The administrator can change the profile image and banner image 
* Any blog/post/tweet/feed will be easy to share with the admin.
* Can directly send messages to followers 
* The reader/blogger can share the content of the blog on his timeline

== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install of a MWB Twitter toolkit for wordpress -Tweets, Feeds, Timeline, posts, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "MWB Twitter toolkit for wordpress -Tweets, Feeds, Timeline, posts" and click Search Plugins. Once you've found our MWB Twitter toolkit for wordpress -Tweets, Feeds, Timeline, posts plugin you can view details about it such as the point release, rating, and description. Most importantly, of course, you can install it by simply clicking "Install Now".

= Manual installation =

The manual installation method involves downloading our MWB Twitter toolkit for wordpress -Tweets, Feeds, Timeline, posts and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.
